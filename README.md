# AI Zork I - Web Edition

An AI-powered reimagining of the classic text adventure "Zork I: The Great Underground Empire". This project uses Google's **Gemini 2.5 Flash** to simulate the game logic and narrative, and **Gemini 2.5 Flash Image** (Nano Banana) to dynamically generate visual representations of the scenes in real-time.

It features a fully responsive, retro-styled terminal interface built with React and Tailwind CSS.

## âœ¨ Key Features

*   **AI Game Master**: Powered by Google Gemini 2.5 Flash. It acts as the game engine, tracking inventory, location, and puzzles while maintaining the witty tone of the original Zork.
*   **Visual Imagination**: Uses `gemini-2.5-flash-image` to generate retro-style pixel art scenes for important locations and events.
*   **Bilingual Support**: Toggle between **English** and **Japanese** instantly. The AI translates the experience in real-time while keeping the game state intact.
*   **Retro Aesthetics**: CRT scanlines, phosphor glow effects, and a custom `VT323` font terminal interface.
*   **Immersive Audio**: Web Audio API-based procedural sound effects for typing, data processing, and dramatic events.
*   **Save & Load System**: 5 local storage slots to save your progress, complete with metadata (location, score, timestamp).
*   **Smart Suggestions**: Context-aware command suggestions generated by the AI to help guide your adventure.
*   **Mobile Optimized**: Responsive design with a collapsible status sidebar and touch-friendly controls.
